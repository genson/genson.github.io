О чём можно забыть в стилях для OK.RU

  • Об ограничение на количестве селектором в одном файле *.css ( до 4096 )
  • Об ограничиние на количество @import в одном файле *.css ( до 32 )
  • Fallback'и для rgba()
  • контекстные классы .ie8, ie9

Польза для odnoklassniki-web

  1. Количество селекторов снизится на 4% ( с 37172 до 35639 )
  2. Суммарный размер стилей станет на 5% меньше ( с 2832 до 2696 Кб)
  3. уйдет ~ -50 картинок ( fader, fill и т.д. )

  • border-radius
  • rgba(), hsla()
  • text-shadow
  • background-size, background-clip, background-origin
  • все CSS3 селекторы
